# Thread-NextJs_Project


## Initializing
```
npx create-next-app@latest          <--- for initializing nextjs

cd <project-name>

npm install @clerk/nextjs           <--- for authentication

npm install @uploadthing/react uploadthing     <--- for file upload

npm install mongoose                <--- for database

npm install svix                    <--- for webhook

npm install tailwindcss-animate 
```
